# Phase 3 - Autonomous PR Follow-Up

## Goal

Make PR outreach and follow-up autonomous.

## Scope

- Media list enrichment
- Personalized pitch variants
- Follow-up scheduler
- Reply classification
- CRM update suggestions
- Campaign performance digest

## Success Criteria

- The agent can run a launch campaign from brief to follow-up report.
- Human approval is only needed for sensitive outbound messages.
