# Phase 1 - Campaign Pack MVP

## Goal

Generate a complete multi-channel campaign pack from one product event.

## Scope

- Product/event brief input
- Audience and launch date fields
- AI generation for:
  - Instagram caption and image prompt
  - Telegram post
  - Bluesky/X short posts
  - press release angle
  - pitch email
  - seven-day campaign calendar

## Success Criteria

- One brief turns into a full launch kit.
- Every artifact has a target channel and publish status.
- Missing AI keys return deterministic demo output.
