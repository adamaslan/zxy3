🎓 KEY INSIGHTS
═══════════════════════════════════════════════════════════════════════════

1. Multi-location galleries are tricky
   → Location counts must always match country counts
   
2. Website availability is volatile
   → 23% of galleries had connectivity issues at audit time
   → Regular verification needed (quarterly minimum)

3. Research is manual but systematic
   → Use templates to stay organized
   → Document sources for transparency

4. Instagram is foundational for artists
   → Enables discovery, engagement tracking, audience analysis
   → Should be one of the first enrichment priorities

5. Enrichment is additive, not destructive
   → Can always add more data later
   → Better to be careful than to overwrite good data
